# Flex Property Directory

A custom WordPress property directory built as a technical assignment.

## Features

- Custom Post Type for Properties
- Custom Taxonomy for Amenities
- AJAX Property Filtering
- GSAP Animations
- Responsive Design
- Sticky Header
- WooCommerce Site Visit Fee Integration

## Technologies Used

- WordPress
- Bootstrap 5
- GSAP
- jQuery AJAX
- ACF Free
- WooCommerce

## Required Plugins

- Advanced Custom Fields
- WooCommerce

## Setup Instructions

1. Install WordPress
2. Activate the custom theme
3. Install required plugins
4. Add ACF fields
5. Create properties and amenities
